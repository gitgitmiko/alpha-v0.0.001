# Kebijakan Privasi (Placeholder)

**Xbox Region Store Browser** — pembaruan terakhir: 2026

## Ringkasan

Aplikasi ini **bukan** produk resmi Microsoft atau Xbox.

## Data yang dikumpulkan

- Preferensi region dan tema (disimpan lokal)
- Wishlist, favorit, riwayat harga (disimpan lokal di perangkat)
- Token FCM (jika notifikasi diaktifkan)
- Data analitik anonim (opsional, Firebase Analytics)
- ID iklan (Google AdMob) jika iklan tidak dinonaktifkan

## Penggunaan data

- Menampilkan harga dan katalog game
- Notifikasi penurunan harga (jika diaktifkan)
- Iklan dan pembelian in-app "Hapus Iklan"

## Pihak ketiga

- Microsoft Store API (displaycatalog.mp.microsoft.com, reco-public.rec.mp.microsoft.com)
- Google AdMob, Google Play Billing
- Firebase Cloud Messaging / Analytics
- exchangerate.host (konversi mata uang)
